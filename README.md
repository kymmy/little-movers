# Le Village — Little Movers

**English Movement & Confidence Club** for children aged 4 months to 6 years.

**Live:** https://little-movers.le-village-marseille.fr

**Portal:** https://le-village-marseille.fr

---

## What is it?

Little Movers is a local project in Marseille 11/12 offering structured movement classes in English for young children. Sunday morning sessions combining yoga, babywearing dance, and motor skills courses in small premium groups.

Currently a dossier/information app presenting the full MVP plan, business strategy, and municipal partnership proposal. Future: interactive booking and community features.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript (strict)
- Tailwind CSS 4
- Framer Motion
- Bilingual FR/EN

## Development

```bash
npm install
npm run dev    # http://localhost:3000
```
